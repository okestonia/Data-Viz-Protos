# Äriregistri aadresside postitus

Siin kaustas on postitusega ["Eesti TOP 10 populaarsemat ettevõtete aadressi"](https://opendata.riik.ee/blogi/Eesti-TOP-10-populaarsemat-ettevotete-aadressi/) seotud failid.

Postituses kasutan Äriregistri [avaandmeid](https://opendata.riik.ee/andmehulgad/ariregister/), 21.11.18 seisuga, EMTA [maksuvõlgnike andmeid](https://www.emta.ee/et/kontaktid-ja-ametist/maksulaekumine-statistika/maksuvolglaste-nimekiri), 02.11.18 seisuga, ja Ehitusregistri [avaandmeid](https://avaandmed.ehr.ee), 24.11.18 seisuga.

Selle analüüsi teeb keeruliseks see, et igal aadressil ei ole ainult 1 hoone. Kui aadressil on mitu hoonet, siis kasutan suurima ehitisaluse pindalaga hoonet. Vahel on ühel hoonel mitu aadressi (nt Uus-Sadama 21/23/25). Osad aadressid on ainult küla täpsusega.
